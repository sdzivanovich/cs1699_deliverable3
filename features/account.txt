Feature:
As a user
I want be able to create and use an account
So that I may communicate with other Wikipedia users

Scenario: Correct Login
Given a valid username
And the corresponding password
And the user is on the login page
When the user enters the username
And the user enters the password
And clicks the login button
Then the user should be logged in
And taken to the main page

Scenario: Viewing user page
Given the name of a user on wikpedia
When a user searches for their name
Then the user should see the user page for that user

Scenario: Editing user page
Given a logged in user
And this user's user page has content
When the user edits their user page
Then the content on their user page should be updated

Scenario: Viewing user contributions
Given the name of a user
When a user views this user's contributions page
Then the user should see the contributions made by this user

Scenario: Posting message to user
Given the name of a user
When a user navigates to this user's page
And clicks on the Talk link
And clicks on the Edit link
Then the user should be able to modify the talk page
